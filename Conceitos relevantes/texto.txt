
OPERADORES RELACIONAIS
  >  MAIOR
  >= MAIOR OU IGUAL
  <  MENOR
  <= MENOR OU IGUAL
  === IGUALDADE
  != DIFERENTE
 
 
Conversão de dados
  1º Texto para número inteiro - parseInt()
  2º Texto para número real - parseFloat()
  3º Número para texto - toString()

Formas de exibir mensagem
   Exibir mensagem via body
   document.write("Hello World!");

   Exibir mensagem via modal (alerta)
   alert("Hello World 2!");

Operadores lógicos
    E   -> &&
    OU  -> ||
    NÃO -> !
